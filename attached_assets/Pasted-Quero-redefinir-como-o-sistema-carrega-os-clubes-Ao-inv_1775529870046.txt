O fluxo correto a implementar:

Para cada liga da lista acima, buscar os times via GET /teams?league={id}&season=2025
Montar a lista completa de clubes do sistema a partir dessas respostas
Cachear essa lista no localStorage por 30 dias (muda pouco)
A navegação por ligas usa exatamente essa estrutura — o usuário vê as ligas, clica, vê os times daquela liga
Ao selecionar um time, buscar o elenco via GET /players/squads?team={id}
A API msmc.cc continua apenas como fallback para dados de OVR e atributos do FC26 — nunca para definir quais clubes existem

Isso garante que o sistema tem apenas times reais do EA FC 26, com elencos atualizados de abril de 2026, sem nenhum time fora do jogo aparecendo.
