CONTEXTO DO PRODUTO
FC Career Manager é uma aplicação web para gerenciar o modo carreira do EA Sports FC (antigo FIFA). É uma plataforma para jogadores que querem viver a experiência de forma imersiva, com dados reais, histórico detalhado e uma camada de narrativa gerada por IA. É gratuita para começar.
MISSÃO DO PROMPT
Crie uma landing page completa em HTML/CSS/JS puro (arquivo único), altamente otimizada para conversão, visualmente extraordinária, com animações envolventes mas elegantes. A página deve fazer o visitante sentir que está entrando em algo especial — não um app genérico, mas uma experiência de futebol de outro nível.

ANIMAÇÕES E INTERAÇÕES — DETALHAMENTO
Animações de entrada (scroll-triggered)
Usar IntersectionObserver para detectar quando elementos entram na viewport. Ao entrar:

Cards: opacity: 0 → 1 + translateY(20px → 0) com transition: 0.6s ease
Títulos de seção: opacity: 0 → 1 + translateX(-20px → 0)
Usar delay escalonado para grupos de cards (0ms, 100ms, 200ms, etc.)

Efeito parallax suave
O fundo do hero (grid/partículas) move a uma velocidade diferente do conteúdo ao rolar. Implementar com window.addEventListener('scroll', ...) atualizando transform: translateY() do background.
Cursor personalizado
Substituir o cursor padrão por:

Círculo pequeno (12px) que segue o mouse com transition: 0.1s
Segundo círculo maior (40px) que segue com delay maior (transition: 0.3s)
Ao hover em botões: ambos os círculos aumentam e ficam roxos

Hover nos cards de feature
css.feature-card:hover {
  transform: translateY(-4px);
  border-color: rgba(124, 92, 252, 0.4);
  background: #1a1a28;
  box-shadow: 0 20px 60px rgba(124, 92, 252, 0.1);
  transition: all 0.3s cubic-bezier(0.34, 1.56, 0.64, 1);
}
Efeito "glow" nos botões principais
css.btn-primary:hover {
  box-shadow: 0 0 30px rgba(124, 92, 252, 0.5), 0 0 60px rgba(124, 92, 252, 0.2);
  transform: translateY(-2px) scale(1.02);
}
Badge animado (hero)
Borda do badge rotaciona com gradient:
css@keyframes rotateBorder {
  from { --angle: 0deg; }
  to { --angle: 360deg; }
}
Usar conic-gradient com CSS custom property animada.
Typing effect na seção de IA
O texto da notícia gerada aparece caractere por caractere com setInterval. Cursor piscante no final. Após terminar, pausa 3s e recomeça com texto diferente (array de 3 notícias rotacionando).
Contador animado (stats)
javascriptfunction animateCounter(el, target, duration) {
  let start = 0;
  const step = target / (duration / 16);
  const timer = setInterval(() => {
    start += step;
    if (start >= target) { start = target; clearInterval(timer); }
    el.textContent = Math.floor(start).toLocaleString('pt-BR') + (el.dataset.suffix || '');
  }, 16);
}

REQUISITOS TÉCNICOS

Arquivo único HTML — todo CSS e JS inline
Google Fonts: importar Bebas Neue, DM Sans, JetBrains Mono
Zero dependências externas de JS — tudo vanilla
Mobile-first responsivo — breakpoints: 768px (tablet), 1024px (desktop)
Performance: usar will-change: transform apenas em elementos animados, requestAnimationFrame para animações JS
Scroll suave: scroll-behavior: smooth no HTML
Todos os links CTA apontam para # (placeholder) com event.preventDefault()

Direto, confiante, esportivo — não corporativo

PT-BR brasileiro, não lusitano


O QUE EVITAR

❌ Backgrounds brancos ou claros
❌ Gradientes roxo-rosa genéricos de IA
❌ Cards com sombra exagerada tipo Material Design
❌ Fontes Inter, Roboto, Arial
❌ Ícones emoji como substituto de SVG
❌ Animações de loading desnecessárias
❌ Pop-ups, modais ou banners de cookie no hero
❌ Muito texto em qualquer seção — cada seção tem 1 mensagem central
